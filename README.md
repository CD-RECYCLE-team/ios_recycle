## Overview
머신러닝 이미지 분류 모델과 공공데이터, 지도 API를 활용한 쓰레기 분류 iOS 어플 개발

[ML model](https://github.com/tlaalsqh7414/TrashClassificationModel).

## Requirements
1. cocoapods 설치  
>  sudo gem install cocoapods

2. 프로젝트 폴더에서 터미널 열고 pod install
> cd path/ios_recycle/CD_RECYCLE
> 
> pod install
